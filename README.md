# anonys_core

Core runtime library for the Anonys C++ finite state machine framework.

This repository provides the state machine engine (`FsmCore`), timer support, tracing interface, and supporting types. It is designed to be consumed as a git submodule of the main Anonys repository, which contains the documentation, code generator, and working examples.
